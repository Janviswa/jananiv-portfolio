## Overview

BloodPrint ID is a full-stack AI web application that classifies fingerprint patterns (Loop, Whorl, and Arch) from uploaded images using an **EfficientNetB0** deep learning model trained on the **SOCOFing dataset (~6,000 images)**, achieving **86% validation accuracy on 1,199 test images**. Based on findings from **four published dermatoglyphic research studies**, the system provides statistical blood group correlations with confidence scoring and generates detailed PDF analysis reports. Designed for educational and research purposes, the platform is deployed on **Vercel and Render**.

## Problem

Fingerprint patterns have been studied for decades for their potential relationship with blood groups, but existing research findings are scattered across academic literature and difficult to explore interactively. There is a need for a user-friendly platform that can analyze fingerprint patterns and visualize these statistical correlations in an accessible way.

## Solution

BloodPrint ID addresses this challenge through a computer vision pipeline powered by an **EfficientNetB0-based deep learning model** trained on ~6,000 SOCOFing fingerprint images. The system processes uploaded fingerprint images, classifies patterns with **86% validation accuracy**, extracts quality metrics (clarity, ridge density, and edge ratio), and generates blood group likelihood estimates with confidence scoring based on four published dermatoglyphic studies. The model is optimized using **TensorFlow Lite** for efficient deployment, with results presented through an intuitive web interface featuring downloadable PDF reports and analysis history tracking.

## Key Features

- **Fingerprint Pattern Classification** — Classifies fingerprints into Loop, Whorl, and Arch patterns using EfficientNetB0 trained on ~6,000 SOCOFing images, achieving 86% validation accuracy on 1,199 test images.
- **Blood Group Correlation Analysis** — Provides statistical blood group likelihood with confidence scoring based on findings from four published dermatoglyphic research studies.
- **Image Quality Assessment** — Extracts clarity score, ridge density, and edge ratio metrics for each fingerprint.
- **TFLite Optimized Deployment** — Model converted to TensorFlow Lite for efficient inference in production.
- **PDF Report Generation** — Generates professional downloadable reports for each analysis.
- **Analysis History** — Stores and manages previous fingerprint analyses.
- **Secure Authentication** — JWT-based user authentication and account management.
- **Cloud Deployment** — Frontend hosted on Vercel, backend on Render.

## Challenges and Learnings

One of the primary challenges was maintaining classification accuracy across varying image qualities in the SOCOFing dataset. Optimizing image preprocessing, extracting meaningful quality metrics (clarity, ridge density, edge ratio), and converting the model to TFLite for efficient production deployment required extensive experimentation. The project strengthened skills in computer vision, deep learning, full-stack development, and AI model deployment.

## Outcome

BloodPrint ID successfully classifies fingerprint patterns with **86% validation accuracy** on 1,199 test images and provides research-based blood group correlation insights with confidence scoring derived from four dermatoglyphic studies. The project demonstrates practical integration of computer vision, deep learning, and full-stack development, deployed end-to-end on Vercel and Render.
