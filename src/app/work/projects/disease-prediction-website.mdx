## Overview

Disease Prediction Website is a bilingual (English & Tamil) machine learning-powered healthcare application that predicts **Heart Disease, Diabetes, and Parkinson's Disease** using **Random Forest and SVM** models, achieving **89%+ accuracy**. Users input health parameters in either language and receive real-time predictions with automated PDF health reports generated in both English and Tamil, tested across **50+ patient scenarios**.

## Problem

Early detection of chronic diseases is essential, but many users lack quick and accessible screening tools. Traditional methods are time-consuming, often require expert consultation, and are generally English-centric. There was a need for a **multilingual, intuitive, and automated system** to provide preliminary disease risk assessments.

## Solution

Built a web-based platform using **Random Forest** and **Support Vector Machine (SVM)** models for disease prediction across 3 conditions. The system accepts bilingual input in English and Tamil, translating Tamil entries to English for processing. Users receive probability-based predictions and automated **PDF health reports in both English and Tamil**, validated across **50+ patient scenarios**, delivered through a real-time **Streamlit interface**.

## Key Features

- **Multi-Disease Prediction** — Predicts likelihood of Heart Disease, Diabetes, and Parkinson's Disease using Random Forest and SVM, achieving 89%+ accuracy.
- **Bilingual Support** — Accepts health parameter input in English and Tamil with NLP-based translation for consistent model processing.
- **Automated PDF Reports** — Generates health reports in both English and Tamil summarizing predictions and health insights, tested across 50+ patient scenarios.
- **Interactive UI** — Streamlit-based interface for easy data entry and visualization.
- **Probability Scores** — Displays likelihood for multiple conditions for each query.
- **Real-Time Inference** — Provides instant predictions with immediate visual feedback.

## Technologies Used

- **Python** — Core language for model development and backend logic.
- **Scikit-learn** — Used for training and evaluating Random Forest and SVM classifiers.
- **NLTK** — Natural language preprocessing and Tamil-to-English translation handling.
- **Pandas / NumPy** — Data cleaning, preprocessing, and feature engineering.
- **Streamlit** — Frontend framework for interactive web application and deployment.
- **Pickle** — Model serialization for production-ready deployment.

## Challenges and Learnings

Handling bilingual input while maintaining consistent model accuracy were key challenges. Feature selection, preprocessing, and synonym mapping were critical for reliable disease prediction. Implementing Tamil-to-English NLP translation pipelines and generating accurate PDF reports in both languages required careful integration. This project strengthened skills in **healthcare AI, multilingual NLP, and end-to-end ML deployment**.

## Outcome

The platform achieved prediction accuracies of approximately **88% for Heart Disease**, **92% for Diabetes**, and **89% for Parkinson's Disease**. Automated bilingual PDF health reports tested across **50+ patient scenarios**, combined with real-time predictions, make it a practical tool for early disease screening and healthcare awareness across English and Tamil-speaking communities.
