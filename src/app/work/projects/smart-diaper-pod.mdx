## Overview

AI-Based Smart Diaper Pod is an intelligent infant monitoring system that combines Artificial Intelligence, IoT, and embedded edge computing to provide real-time diaper condition tracking and caregiver alerts. The system uses multiple sensors to monitor moisture, pressure, temperature, and infant movement, enabling proactive hygiene management and improved infant comfort through automated monitoring and intelligent decision-making.

## Problem

Maintaining proper diaper hygiene requires continuous monitoring, which can be challenging for caregivers, especially during nighttime or when managing multiple responsibilities. Delayed diaper changes may lead to discomfort, skin irritation, infections, and disrupted sleep patterns. Traditional manual checking methods are time-consuming and often unreliable.

## Solution

Designed and developed a smart wearable monitoring system consisting of a Sense Pod and Core Pod architecture. The system collects real-time data from multiple sensors, processes it using an embedded machine learning model running on an ESP32 microcontroller, and classifies diaper conditions such as dry, wet, stool indication, and abnormal temperature. Caregivers receive instant alerts and monitoring insights through a connected mobile application.

## Key Features

- **Real-Time Monitoring**: Continuously tracks diaper wetness, temperature, pressure, and infant movement.
- **AI-Based Classification**: Uses embedded machine learning to intelligently identify diaper conditions.
- **Multi-Sensor Fusion**: Combines data from multiple sensors to improve detection accuracy and reliability.
- **Instant Caregiver Alerts**: Sends real-time notifications through a mobile application when attention is required.
- **Edge Intelligence**: Performs on-device processing for low latency, improved privacy, and reduced cloud dependency.
- **Cloud Connectivity**: Synchronizes monitoring data for remote access and historical tracking.

## Technologies Used

### Hardware
- **ESP32-C3** — Embedded processing and wireless communication
- **Moisture Sensor** — Wetness detection
- **Force Sensor** — Stool indication detection
- **Temperature Sensor** — Body temperature monitoring
- **3-Axis Accelerometer Sensor** — Movement and discomfort tracking
- **Custom PCB** — Compact hardware integration
- **Li-Ion / Li-Po Battery** — Portable power supply
- **Wireless Charging Module** — Contactless charging support

### Software
- **Python** — Machine learning model development and data processing
- **Random Forest** — Diaper condition classification
- **Firebase** — Cloud database and real-time synchronization
- **React Native** — Mobile application development
- **Tailwind CSS** — Responsive user interface design
- **KiCad** — PCB design and hardware development
- **Arduino IDE** — ESP32 firmware development
- **Embedded C/C++** — Sensor integration and device control

## Challenges

Designing a reliable multi-sensor monitoring system while maintaining infant safety, comfort, and low power consumption was a significant challenge. Optimizing the machine learning model for embedded deployment and minimizing false alerts required extensive sensor calibration, feature engineering, and real-world testing.

## Results

Successfully developed a functional prototype capable of monitoring diaper conditions in real time and delivering instant caregiver notifications. The system demonstrated reliable detection of wetness, temperature abnormalities, and infant activity while reducing the need for frequent manual checks.
