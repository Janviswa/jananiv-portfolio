## Overview

CineMatch is an AI-powered movie recommendation system that generates personalized movie recommendations using **TF-IDF vectorization and cosine similarity** on the **TMDB 5000 dataset**, achieving **87% recommendation accuracy**. By analyzing genres, cast, crew, keywords, and movie descriptions, the system enriches discovery with real-time posters, IMDb/TMDB ratings, trailers, and detailed movie information through **TMDB and OMDb API integration** via an interactive Streamlit interface.

## Problem

With thousands of movies available across streaming platforms, finding relevant content can be overwhelming and time-consuming. Users often struggle to discover movies that match their preferences, highlighting the need for an intelligent recommendation system that simplifies movie exploration and improves content discovery.

## Solution

CineMatch addresses this challenge using a content-based recommendation engine built on **TF-IDF vectorization and cosine similarity**. The system processes movie metadata from the **TMDB 5000 dataset**, creates feature-rich representations from genres, cast, crew, and descriptions, and identifies similar movies based on content characteristics — achieving **87% recommendation accuracy**. Integration with **TMDB and OMDb APIs** provides real-time posters, IMDb/TMDB ratings, trailers, and additional movie details through an interactive Streamlit interface.

## Key Features

- **Smart Movie Recommendations** — Suggests similar movies using TF-IDF vectorization and cosine similarity on the TMDB 5000 dataset, achieving 87% recommendation accuracy.
- **TMDB & OMDb API Integration** — Provides real-time movie posters, IMDb/TMDB ratings, trailers, and detailed movie information for an enriched discovery experience.
- **Movie Carousel** — Interactive banner showcasing popular movies.
- **Dual Rating System** — Displays ratings from both TMDB and IMDb.
- **Trailer Integration** — Provides direct access to movie trailers through YouTube links.
- **Rich Movie Cards** — Displays posters, release year, overview, and ratings.
- **Hover Effects** — Interactive poster effects revealing movie descriptions.
- **Responsive UI** — Streamlit-based interface optimized for seamless user interaction.

## Technologies Used

### Machine Learning & Recommendation Engine

- **Python** — Core programming language for recommendation logic and application development.
- **Scikit-learn** — Implements TF-IDF vectorization and cosine similarity calculations.
- **Pandas** — Data preprocessing, cleaning, and feature engineering.
- **NumPy** — Numerical computations and matrix operations.

### APIs & Data Sources

- **TMDB API** — Retrieves real-time movie posters, metadata, ratings, and trailers.
- **OMDb API** — Provides IMDb ratings and additional movie information.
- **TMDB 5000 Dataset** — Primary dataset used for recommendation generation.

### Frontend & Deployment

- **Streamlit** — Interactive web application framework and deployment platform.
- **Requests** — API communication and external data retrieval.

## Challenges and Learnings

One of the primary challenges was achieving reliable recommendation accuracy while efficiently processing large amounts of movie metadata from the TMDB 5000 dataset. Building feature-rich movie representations from genres, cast, crew, and descriptions and optimizing TF-IDF similarity calculations required careful preprocessing and feature selection. The project strengthened skills in recommendation systems, NLP-based text processing, API integration, and interactive web application development.

## Outcome

CineMatch successfully delivers personalized movie recommendations with **87% recommendation accuracy**, enriched with real-time posters, IMDb/TMDB ratings, and trailers through TMDB and OMDb API integration. The system demonstrates the effectiveness of content-based filtering for movie discovery while providing an engaging user experience through an intuitive Streamlit interface.
